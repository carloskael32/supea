PROCESOS
Las opciones de Productos, Vendedores y Proveedores, son importantes para la configuración de los procesos del negocio, en cada uno de estos procesos es importante definir adecuadamente el código clave. El proceso de Altas, Bajas, Modificaciones; es importante para el registro y eliminación de información en cada uno de estos módulos.

Ventas y Devoluciones, corresponde a los procesos diarios, de entrega y devolución de productos a vendedores. En cada proceso de ventas y devoluciones, existe un código único de identificación del proceso. El registro de ventas y devoluciones, se registra por día de proceso. El proceso que concentra la información diaria, es el reporte de Entregas versus Devoluciones.

La opción de liquidación, corresponde a los pagos efectuados por los vendedores y también el pago de recibos por créditos. La clave de acceso es el código de vendedor y la fecha de proceso, en caso de pago de entregas versus devoluciones. Para el pago de recibos por créditos, cada recibo tiene un código único de identificación.
